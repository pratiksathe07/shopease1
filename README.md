# ShopEase 🛒

A full-stack E-Commerce Web Application built using the MERN Stack.

## 🚀 Live Demo

https://shopease1-frontend-nyaf.onrender.com/

## 🛠️ Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication
* Render Deployment

## ✨ Features

* User Authentication
* Product Search & Categories
* Wishlist System
* Shopping Cart
* Order Management
* Admin Dashboard
* Product Management
* Responsive UI
* REST APIs
* MongoDB Database Integration


## 📂 Project Structure

* frontend/
* backend/

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/pratiksathe07/shopease1.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```
